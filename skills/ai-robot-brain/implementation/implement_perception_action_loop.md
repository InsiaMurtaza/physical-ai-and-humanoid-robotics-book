# Implement Perception-Action Loop Skill

## Purpose
Implement closed-loop perception-action system for AI-powered robot brain.

## Parameters
- loop_rate: Rate of perception-action cycle
- sensor_fusion: Types of sensors to fuse
- action_types: Categories of actions to execute

## Output
Implementation with:
- Perception pipeline
- Decision-making module
- Action execution interface
- Feedback mechanisms
- Safety checks and timeouts
