# Write Perception Pipeline Documentation Skill

## Purpose
Create comprehensive documentation for AI-based perception pipelines in robot brain systems.

## Parameters
- pipeline_component: Specific perception component (object detection, segmentation, etc.)
- framework: Target framework (Isaac ROS, OpenCV, etc.)
- use_case: Specific application domain

## Output
Pipeline documentation with:
- Architecture overview
- Processing steps
- Configuration parameters
- Calibration procedures
- Performance tuning guidelines
