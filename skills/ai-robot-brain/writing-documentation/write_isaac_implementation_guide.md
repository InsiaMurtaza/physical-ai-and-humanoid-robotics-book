# Write Isaac Implementation Guide Skill

## Purpose
Create detailed implementation guides for NVIDIA Isaac-based robot brain systems.

## Parameters
- guide_type: Type of guide (setup, integration, optimization)
- component: Specific Isaac component (Sim, ROS, perception stack)
- requirements: Hardware or software requirements

## Output
Implementation guide with:
- System requirements
- Installation procedures
- Configuration steps
- Testing procedures
- Best practices and recommendations
