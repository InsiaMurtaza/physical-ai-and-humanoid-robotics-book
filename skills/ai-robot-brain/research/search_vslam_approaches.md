# Search VSLAM Approaches Skill

## Purpose
Search and compare Visual SLAM approaches available in Isaac Sim and related robotics frameworks.

## Parameters
- algorithm: Specific VSLAM algorithm (ORB-SLAM, RTAB-MAP, etc.)
- platform: Target platform (Isaac Sim, ROS2, etc.)
- constraints: Computational or accuracy constraints

## Output
Comparison summary with:
- Algorithm characteristics
- Performance benchmarks
- Implementation details
- Integration requirements
- Trade-offs and considerations
