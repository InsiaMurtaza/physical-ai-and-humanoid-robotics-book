# Extract AI Robot Brain Architectures Skill

## Purpose
Identify and summarize common architectures for AI-powered robot brains using NVIDIA Isaac ecosystem.

## Parameters
- capability: Primary capability (navigation, manipulation, perception)
- hardware: Target hardware (Jetson, Xavier, etc.)
- performance: Performance requirements (real-time, latency, etc.)

## Output
Architecture summary with:
- Processing pipeline designs
- Sensor fusion approaches
- Decision-making patterns
- Learning integration methods
- Resource optimization strategies
