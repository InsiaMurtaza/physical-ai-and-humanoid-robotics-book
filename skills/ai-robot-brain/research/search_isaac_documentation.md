# Search NVIDIA Isaac Documentation Skill

## Purpose
Search and extract relevant information from NVIDIA Isaac documentation, tutorials, and resources for AI-powered robot brain implementations.

## Parameters
- query: Search term related to Isaac Sim or Isaac ROS
- component: Specific Isaac component (Omniverse, ROS bridge, perception, etc.)
- focus_area: Specific aspect (navigation, manipulation, perception)

## Output
Structured summary with:
- Key concepts and definitions
- Relevant code snippets
- Best practices
- Hardware requirements
- Integration patterns
