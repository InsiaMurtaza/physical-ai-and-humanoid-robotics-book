# Explain Isaac Sim Pipeline Skill

## Purpose
Explain Isaac Sim and Isaac ROS pipelines including VSLAM, navigation, and synthetic data generation.

## Parameters
- pipeline_component: Specific component (VSLAM, navigation, perception, etc.)
- use_case: Primary use case (training, validation, deployment)
- complexity: Level of technical detail

## Output
Pipeline explanation with:
- Component architecture
- Data flow diagram
- Key parameters and configurations
- Performance considerations
- Integration points with ROS 2
