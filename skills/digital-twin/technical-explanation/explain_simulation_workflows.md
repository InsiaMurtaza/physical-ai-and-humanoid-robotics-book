# Explain Simulation Workflows Skill

## Purpose
Explain physics simulation, environment building, and sensor simulation workflows in digital twin systems.

## Parameters
- workflow_type: Type of workflow (physics setup, environment creation, sensor config)
- platform: Target platform (Gazebo, Unity, Isaac Sim)
- complexity: Level of detail required

## Output
Detailed explanation with:
- Step-by-step process
- Key parameters and configurations
- Common pitfalls and solutions
- Performance considerations
- Diagram representation (Mermaid/ASCII)
