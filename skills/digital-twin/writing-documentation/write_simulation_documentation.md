# Write Simulation Documentation Skill

## Purpose
Create clear, comprehensive documentation for digital twin simulation setups and workflows.

## Parameters
- document_type: Type of document (tutorial, reference, troubleshooting)
- audience: Target audience (beginner, intermediate, expert)
- scope: Specific aspect to document (setup, configuration, optimization)

## Output
Documentation with:
- Clear objectives and prerequisites
- Step-by-step instructions
- Configuration examples
- Troubleshooting tips
- Performance guidelines
