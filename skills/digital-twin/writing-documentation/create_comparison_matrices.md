# Create Comparison Matrices Skill

## Purpose
Generate structured comparison matrices for different digital twin technologies and approaches.

## Parameters
- comparison_type: What to compare (Gazebo vs Unity, physics engines, etc.)
- criteria: Evaluation criteria to include
- depth: Level of technical detail

## Output
Comparison matrix with:
- Side-by-side feature analysis
- Performance benchmarks
- Use case recommendations
- Pros and cons for each option
- Decision guidance
