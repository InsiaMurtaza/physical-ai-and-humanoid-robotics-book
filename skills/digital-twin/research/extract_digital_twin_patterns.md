# Extract Digital Twin Patterns Skill

## Purpose
Identify and summarize common patterns in Digital Twin implementations across robotics simulation platforms.

## Parameters
- platform: Target platform (Gazebo, Unity, Isaac Sim, etc.)
- use_case: Specific application area (training, validation, monitoring)
- requirements: Functional and non-functional requirements

## Output
Pattern summary with:
- Architecture patterns
- Data synchronization methods
- Sensor simulation approaches
- Physics fidelity considerations
- Real-time constraints
