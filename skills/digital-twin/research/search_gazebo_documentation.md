# Search Gazebo Documentation Skill

## Purpose
Search and extract relevant information from Gazebo documentation, tutorials, and resources for Digital Twin applications.

## Parameters
- query: Search term related to Gazebo simulation
- version: Gazebo version to focus on (optional)
- focus_area: Specific aspect (physics, sensors, plugins, etc.)

## Output
Structured summary with:
- Key concepts and definitions
- Relevant code snippets
- Best practices
- Version-specific considerations
