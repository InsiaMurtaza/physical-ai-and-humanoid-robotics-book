# Search Unity Robotics Documentation Skill

## Purpose
Search and extract relevant information from Unity Robotics documentation, packages, and resources for Digital Twin applications.

## Parameters
- query: Search term related to Unity robotics
- package: Specific Unity package (URP, ROS-TCP-Connector, etc.)
- focus_area: Specific aspect (physics, environment, sensor simulation, etc.)

## Output
Structured summary with:
- Key concepts and definitions
- Relevant code snippets
- Integration patterns
- Performance considerations
