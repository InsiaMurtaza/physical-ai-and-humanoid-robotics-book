# Setup Unity Robotic Simulation Skill

## Purpose
Implement Unity-based robotic simulations with appropriate physics, sensors, and environment assets.

## Parameters
- simulation_type: Type of simulation (kinematic, dynamic, realistic)
- robot_prefab: Target robot prefab for simulation
- sensor_requirements: List of sensors to simulate

## Output
Implementation with:
- Scene setup
- Robot integration
- Sensor components
- Physics configuration
- ROS connection setup
