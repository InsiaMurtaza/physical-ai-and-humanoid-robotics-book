# Search Multimodal Integration Methods Skill

## Purpose
Search and summarize methods for integrating vision, language, and action modalities in robotic systems.

## Parameters
- modality_focus: Primary integration focus (vision-language, language-action, etc.)
- framework: Target framework (PyTorch, TensorFlow, etc.)
- application: Specific application area

## Output
Integration summary with:
- Fusion techniques
- Architecture patterns
- Training methodologies
- Performance considerations
- Implementation challenges
