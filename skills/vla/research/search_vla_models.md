# Search VLA Models Skill

## Purpose
Search and extract information about Vision-Language-Action models and their implementations.

## Parameters
- model_type: Specific VLA model family (RT-1, BC-Z, etc.)
- domain: Application domain (household, industrial, etc.)
- requirements: Technical requirements (compute, data, etc.)

## Output
Model summary with:
- Architecture details
- Training methodology
- Performance benchmarks
- Implementation requirements
- Integration patterns
