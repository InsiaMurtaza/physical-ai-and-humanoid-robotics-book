# Extract VLA Workflows Skill

## Purpose
Identify and document common workflows for Vision-Language-Action systems in robotics.

## Parameters
- input_type: Primary input modality (image, text, sensor data)
- action_space: Target action space (navigation, manipulation, etc.)
- complexity: Task complexity level

## Output
Workflow summary with:
- Processing pipeline
- Decision points
- Error handling
- Feedback mechanisms
- Performance optimization strategies
