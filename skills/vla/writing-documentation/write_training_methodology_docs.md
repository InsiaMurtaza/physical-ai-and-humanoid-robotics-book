# Write Training Methodology Documentation Skill

## Purpose
Document training methodologies for Vision-Language-Action models in robotics applications.

## Parameters
- training_type: Type of training (supervised, reinforcement, imitation)
- data_modality: Modalities involved in training
- evaluation: Evaluation metrics and procedures

## Output
Training methodology documentation with:
- Data preparation procedures
- Model architecture
- Training procedure
- Evaluation methodology
- Validation criteria
