# Write Multimodal System Specifications Skill

## Purpose>
Create detailed technical specifications for multimodal AI systems that integrate vision, language, and action.

## Parameters
- specification_type: Type of spec (functional, non-functional, interface)
- integration_level: Level of system integration (component, subsystem, system)
- compliance: Standards or requirements to address

## Output
Technical specification with:
- System requirements
- Interface definitions
- Performance criteria
- Quality attributes
- Validation criteria
