# Generate Robot Architecture Diagrams Skill

## Purpose
Generate clean, readable diagrams (Mermaid/ASCII) for robot architectures involving VLA systems.

## Parameters
- architecture_type: Type of architecture (perception, control, multi-modal)
- complexity: Level of detail in the diagram
- format: Preferred output format (Mermaid, ASCII, etc.)

## Output
Architecture diagram with:
- Component relationships
- Data flow directions
- Interface specifications
- Layered abstraction levels
- Integration points
