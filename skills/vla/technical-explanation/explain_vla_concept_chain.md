# Explain VLA Concept Chain Skill

## Purpose
Convert VLA concepts into structured explanation chains (voice → language → planning → ROS 2 actions).

## Parameters
- input_modality: Primary input (voice, text, image)
- action_type: Target action category (navigation, manipulation, etc.)
- complexity: Level of technical detail

## Output
Concept chain explanation with:
- Multi-modal processing steps
- Language understanding pipeline
- Action planning sequence
- ROS 2 command generation
- Feedback and correction mechanisms
