# Citation Guide for APA Format

This guide explains how to properly cite sources in APA format throughout the book.

## In-text Citations

Use the following format for in-text citations:

- Single author: (Author, Year) or Author (Year)
- Multiple authors: (Author1 et al., Year) when 3+ authors
- Direct quotes: (Author, Year, p. #)

## Examples

```markdown
According to Smith (2023), ROS 2 provides improved communication patterns...

The distributed architecture is fundamental to robotics middleware (Smith et al., 2023).

"ROS 2 serves as the nervous system for robots" (Johnson, 2022, p. 45).
```

## Reference Lists

At the end of each chapter, include a reference list in this format:

```markdown
## References

<div class="reference-list">

- Smith, J. A. (2023). *Modern robotics: Systems and applications*. Robotics Journal Publications.

- Smith, J. A., Johnson, M. B., & Williams, R. C. (2023). Distributed communication in robotic systems. *International Journal of Robotics Research*, 42(3), 123-145.

- Johnson, P. L. (2022). Middleware architectures for robotics. In *Proceedings of the IEEE International Conference on Robotics and Automation* (pp. 40-45). IEEE.

</div>
```

## Citation CSS Classes

The following CSS classes are available for styling citations:

- `.apa-citation` - for individual citation entries
- `.reference-list` - for the container of reference lists
- `.reference-item` - for individual reference items
- `blockquote.with-citation` - for blockquotes with citations
- `.citation` - for citation information within blockquotes
